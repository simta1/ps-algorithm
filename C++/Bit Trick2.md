### 부분집합 순회
```cpp
for (int i = mask; i > 0; i = (i - 1) & mask){
    cout << mask << "(" << bitset<N>(mask) << ") " << bitset<N>(i) << "\n";
}

// ex) mask = 11101
// 29(11101) 11101
// 29(11101) 11100
// 29(11101) 11001
// 29(11101) 11000
// 29(11101) 10101
// 29(11101) 10100
// 29(11101) 10001
// 29(11101) 10000
// 29(11101) 01101
// 29(11101) 01100
// 29(11101) 01001
// 29(11101) 01000
// 29(11101) 00101
// 29(11101) 00100
// 29(11101) 00001
```
mask의 부분집합들 중 공집합만 제외하고 전부 순회

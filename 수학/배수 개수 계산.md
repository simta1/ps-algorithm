## 배수 개수 계산
```cpp
const int MAX = *max_element(v.begin(), v.end());
vector<T> cnt(MAX + 1);
for (auto e : v) ++cnt[e];

for (int i = 1; i <= MAX / 2; i++) {
    for (int j = 2 * i; j <= MAX; j += i) cnt[i] += cnt[j];
}
```
### 시간복잡도
$O(NlogN)$   

### 사용설명
배열 `v`에서 `x`의 배수의 개수를 `cnt[x]`에 저장함
